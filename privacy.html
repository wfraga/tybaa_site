Política de Privacidade

A presente Política de Privacidade descreve como o Tybaa Zap coleta, usa e protege as informações dos usuários ao utilizar nossa plataforma de automação via API Oficial do WhatsApp.

1. Coleta de Informações

Coletamos informações estritamente necessárias para a prestação de nossos serviços, incluindo:

  • Dados de Cadastro: Nome, e-mail e dados de faturamento (processados via Asaas).
  • Dados de Mensagens: Conteúdo processado pela API Oficial para fins de automação e resposta inteligente.
  • Metadados: Informações de log para garantir a segurança e estabilidade da integração via n8n e Drupal.

2. Uso da Inteligência Artificial

O Tybaa Zap utiliza modelos de processamento de linguagem natural (IA) para interpretar comandos de voz e texto. As mensagens são processadas de forma efêmera para gerar respostas, não sendo utilizadas para treinamento de modelos externos sem consentimento prévio.

3. Compartilhamento com Terceiros

Para o funcionamento do sistema, os dados podem ser compartilhados com:

  • Meta/Facebook: Provedor da infraestrutura da API Oficial do WhatsApp Business.
  • Asaas: Para processamento seguro de pagamentos e assinaturas.

4. Direitos do Usuário (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (LGPD), o usuário possui o direito de solicitar o acesso, retificação ou exclusão definitiva de seus dados de nossa base Multi-tenant a qualquer momento.

5. Segurança

Empregamos protocolos de criptografia e camadas de segurança em nossa arquitetura Drupal para proteger as informações contra acessos não autorizados.

6. Contato

Dúvidas sobre esta política podem ser enviadas para o encarregado de dados através do suporte oficial do projeto.
